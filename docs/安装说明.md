# 安装说明

## 环境要求

- Node.js 20+
- npm 10+
- Python 3.10+，仅 FastAPI 需要
- Codex CLI，可选，用于安装本地 plugin/skill

## 1. 本地校验协议

```bash
npm run verify:offline
```

这一步不需要安装第三方依赖，只验证 DHP 协议链路是否能跑通。

## 2. 安装依赖

```bash
npm install
```

## 3. 构建并同步 DHP Bundle

```bash
npm run dhp:validate
npm run dhp:build
npm run dhp:sync
```

## 4. 运行 Next.js 前端

```bash
npm run web:dev
```

访问：`http://localhost:3000`

## 5. 运行 Hono API

```bash
npm run api:hono:dev
```

访问：`http://localhost:8787/health`

## 6. 运行 FastAPI，可选

```bash
cd apps/api-fastapi
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

访问：`http://localhost:8000/docs`

## 7. 安装 Codex Plugin

本仓库已经提供 repo marketplace：

```txt
.agents/plugins/marketplace.json
```

在仓库根目录启动 Codex：

```bash
codex
/plugins
```

选择 marketplace：**DHP 本地插件市场**，安装 **DHP Next Shadcn**。

安装后新开对话，输入：

```txt
请使用 design-handoff-next-shadcn，读取 ./handoff/handoff.json，按协议生成或修复 Next.js/shadcn 页面。
```
