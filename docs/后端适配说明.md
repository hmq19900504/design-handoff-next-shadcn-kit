# 后端适配说明

本项目提供两种后端路径。

## 路径 A：Hono + Node.js + TypeScript

适合希望前后端共享 TypeScript 类型的团队。

```bash
npm run api:hono:dev
```

核心文件：

```txt
apps/api-hono/src/index.ts
apps/api-hono/src/schema.ts
apps/api-hono/src/data.ts
```

## 路径 B：FastAPI + Python

适合已有 Python 服务、算法服务或风控模型服务的团队。

```bash
cd apps/api-fastapi
uvicorn app.main:app --reload --port 8000
```

核心文件：

```txt
apps/api-fastapi/app/main.py
```

## API 契约来源

不管使用 Hono 还是 FastAPI，字段和接口都必须来自：

```txt
handoff/api/api-contract.json
```

Agent 不应自行发明字段。
